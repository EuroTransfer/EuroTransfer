Aperçu en direct
Comment publier
  1. Remplis le formulaire (tu peux empiler plusieurs articles dans la file).
  2. Clique Télécharger data.js : le fichier contient tes anciens articles + les nouveaux.
  3. Sur GitHub, ouvre le dossier assets/js/ → Add file → Upload files, dépose le nouveau data.js, puis Commit changes.
  4. GitHub Pages met le site à jour tout seul en ~1 min. C'est en ligne !
💡 Astuce : seul le fichier data.js change quand tu ajoutes des articles.
